# Chandra AI Chatbot 🦞

A premium, professional-grade AI assistant powered by the 4K Agent, capable of high-resolution image generation, voice synthesis, and advanced conversational reasoning.

## 🚀 One-Click Deployment

To fix the "No Reply" issue on GitHub Pages, you must deploy the backend to Render. Click the button below to start:

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/chandramondal-1/chatbot)

### Deployment Instructions:
1.  Click the **Deploy to Render** button above.
2.  In the Render dashboard, it will ask for **Environment Variables**.
3.  Add your keys:
    *   `OPENROUTER_API_KEY`
    *   `NVIDIA_API_KEY`
    *   `DEEPSEEK_API_KEY`
    *   `KIMI_API_KEY`
4.  Once deployed, copy your new **Render URL** (e.g., `https://chatbot-api.onrender.com`).
5.  Paste that URL into the top of `script.js` in your GitHub repo.
